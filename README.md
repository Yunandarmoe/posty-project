# Posty Project

This is a simple project

## Installation

- git clone `https://github.com/Yunandarmoe/posty-project.git`

- Open project folder

- Copy .env file

- Create database same with .env database name

- php artisan migrate

- Install tailwind css 
  - npm install -D tailwindcss
  - npx tailwindcss init

## Finally run

- php artisan serve
